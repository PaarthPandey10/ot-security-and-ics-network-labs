# Central Screenshots Repository

> A centralized storage directory for all visual assets, network topologies, GUI configurations, and CLI outputs across the OT Security Labs.

---

## Overview

This folder serves as the unified media repository for the entire project. To maintain a clean directory structure, all individual lab folders reference images stored here using relative paths.

---

## Usage / How to Use

When contributing or adding new visual guides, place all `.png`, `.jpg`, or `.gif` files in this root `screenshots/` directory. 

Reference them in your markdown files using the standard relative path format:
`![](../screenshots/your-image-name.png)`
